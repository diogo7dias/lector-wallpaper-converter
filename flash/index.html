  1. Open this page in Chrome or Edge on a computer (Web Serial only).
  2. Plug the reader into a USB port with a data cable.
  3. Click Update my reader and pick the reader's serial port when asked.
  4. Wait for the flash to finish. The reader reboots into the new build on its own.
Nothing is uploaded to a server. The firmware file is served from this site and written to the device entirely inside your browser.
How this works

Your Lector reader runs on an ESP32-C3 chip. This page uses esp-web-tools (the same in-browser flasher used by many ESP32 projects) to talk to the chip over the USB serial port via the browser's Web Serial API.

  • Update writes firmware.bin at offset 0x10000 — the app partition — and nothing else, so your data survives.
  • Full rescue writes the bootloader (0x0), partition table (0x8000), OTA data (0xe000), and app (0x10000), then erases the rest. It rebuilds a device from bare metal.
  • One firmware serves both X3 and X4; the running code detects the model at boot.
